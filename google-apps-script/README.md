# Google Sheets 예매 동기화

`Code.gs`를 현재 웹앱 배포 프로젝트에 붙여 넣고 새 버전으로 재배포합니다.

- 실행 계정: 본인
- 액세스 권한: 모든 사용자
- 대상 문서: `1f9gfxWvypp_Pavmp-1usM4ozgvrO2Zhqc6nbKrkbT4o`
- 대상 탭: `07.25 SAT 1PM`, `07.25 SAT 4PM`, `07.26 SUN 1PM`, `07.26 SUN 4PM` (없으면 자동 생성)
- 기존 `예매내역` 탭의 데이터는 일정에 맞는 4개 탭으로 자동 이동한 뒤 기존 탭을 삭제합니다.
- 일정 수정 시 행이 해당 회차 탭으로 이동하고, 예매 취소 시 해당 행이 삭제됩니다.

기존 웹앱 배포를 수정한 뒤 **배포 관리 → 수정 → 새 버전 → 배포**하면 기존 `/exec` 주소를 유지할 수 있습니다.
